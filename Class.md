## 클래스의 기본 구조
* 고유의 특징값을 뜻하는 ```속성```
* 기능을 구현한 ```함수```

### 클래스의 요소
```Kotlin
    fun main(
        val str: String,  // 클래스의 속성들을 선언하는 동시에 생성자를 선언하는 방법이다
        val int: Int
    )
```
* 생성자란 새로운 인스턴스를 만들기 위해 호출하는 함수
>인스턴스의 속성을 초기화 및 인스턴스 생성시 구문을 수행 할 수 있다
* ```ìnit```을 통해 인스턴스를 생성하며 구문을 수행할 수 있다
  * 파라미터나 반환형이 없으며, 생성자를 통해 인스턴스가 만들어질 호출되는 함수이다

* 클래스에 기본적으로 생성되는 생성자 이외에 필요할 때 추가적으로 선언하여 사용할 수 있는 보조 생성자도 존재한다
  * 인스턴스 생성시 추가적인 구문을 수행하는 기능도 있다
```Kotlin
    fun main(
        val str: String,
        val int: Int
    ) {
      constructor(str: String): this(
          str = str, // this라는 키워드를 사용하여 기본 생성자가 필요로 하는 파라미터를 넘겨준다
          int = 10
      ) {
        println("보조 생성자가 사용됨") // 중괄호 안에 실행될 추가적인 구문을 작성하면 된다
      }
    }
```

### 클래스의 상속
* 클래스를 상속받는 이유는 이미 존재하는 클래스에 변수나 함수를 새로 추가하여 만들어야 할때 
* 클래스의 공통점을 추상화하여 뽑아내야 할 때에도 사용한다
> 이때, 상속하는 쪽을 Super Class, 상속받는 쪽을 Sub Class라고 표기한다
* 클래스가 ```open```상태가 아니라면 상속을 받을 수 없다
